Package content

* intogen-sm-virtualmachine.ova

This file contains a virtual machine with all the pipeline installed. Follow the next steps to use it:

         

  1. Install VirtualBox (https://www.virtualbox.org)
  2. Open VirtualBox and go to "Import appliance" menu and import "intogen-sm-virtualmachine.ova"
  3. Start the new virtual machine
  4. You don't need to log into the virtual machine because it run in a server mode, so you only need to point your browser to: http://localhost:5000
  5. Go to the tab "New analysis" and start a new analysis with the example data or your own data.

* intogen-sm-examples

This folder contains several example data to test the pipeline.

* intogen-sm-sourcecode.zip

Here you have the pipeline source code ready to be installed in a linux system with Perl and Python 2.7. You only need to extract the ZIP file and start the web server like this: "./intogen start"
